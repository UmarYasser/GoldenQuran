The population starts at Y=2 and grows in a classic S-shape, asymptotically approaching the stable carrying capacity of K=600.

Interpreting Fixed Points (H=0)

Y* = 0 (Unstable)

This is the extinction fixed point. If the population is exactly 0, it stays 0. But if Y>0, the population grows and moves *away* from this point. It is an unstable equilibrium.

Y* = 600 (Stable)

This is the carrying capacity. If the population starts below 600, it grows *toward* it. If it starts above 600, it would decline *toward* it. It is a stable equilibrium.
